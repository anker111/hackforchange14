Hack for Change 2014: Miami

 

Kickoff event: EMERGE AMERICAS HACKATHON
May 2-4 | University of Miami Life Science & Technology Park, 1951 NW 7th Ave, Miami, FL 33136

Join Code for Miami, a local, volunteer brigade of Code for America, to design, develop, and use technology to improve our community throughout the month of May.

Don't know how to code? You can still hack! Check out our no-code challenges.

Challenges:

National Challenges

National Hack for Change Challenges will be announced at hackforchange.org, and you can still contribute ideas and datasets.

Local Challenges

Local business tax data API

Use the new local business tax data REST API to create visualizations, maps or apps that help Miamians better understand their local economy and their neighborhoods.

How to use the API


311 data visualizations and prototypes

Build visualizations and prototypes that demonstrate how 311 data might be used.

Sample 311 dataset


Open311 REST API wrapper

311 is a system that tracks civic issues in Miami and in communities across the country. Miami-Dade County is gearing up to open its 311 data, and we need help building an Open311 compliant API wrapper for the data.

Open311 data standards

Sample 311 dataset


Permit data visualizations and prototypes

Miami-Dade County has released a sample of its permitting data. Build visualizations and prototypes that demonstrate how permit data might be used.
